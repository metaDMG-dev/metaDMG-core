metaDMG.utils.extract_samples(paths, prefix='', suffix='', long_name=False)[source]¶

Extract all alignment files from a list of files. Truncates the name of the files, controlled by prefix, suffix, and long_name

Parameters

  • paths (list[Path]) – List of paths to be extracted

  • prefix (str) – The prefix to be added to the name, by default “”

  • suffix (str) – The suffix to be added to the name, by default “”

  • long_name (bool) – Whether or not to use the full name, by default False

Return type

Dictionary with names as keys and files as values.
